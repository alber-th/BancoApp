# Sistema de Gestão de Conta Bancária
Este projeto simula um sistema bancário em Java, usando Programação Orientada a Objetos.

Funcionalidades
Criação de contas correntes e poupança

Depósito, saque e transferência

Extrato de contas

Rendimento mensal para poupança

Tratamento de exceções de saldo insuficiente

Estrutura de Pastas

BancoApp/
  └── src/
      ├── excecoes/
      │   └── SaldoInsuficienteException.java
      ├── interfaces/
      │   └── OperacoesBancarias.java
      ├── modelos/
      │   ├── Conta.java
      │   ├── ContaCorrente.java
      │   └── ContaPoupanca.java
      └── BancoApp.java


Como executar
Compile todos os arquivos Java dentro da pasta src

Execute BancoApp.java como classe principal
